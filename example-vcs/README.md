# Universal Resolver Driver: did:btcr2

This contains example Verifiable Credentials and other JSON-LD documents with Data Integrity Proofs, signed by `did:btcr2` DIDs.

Types of JSON-LD documents:
- [ActivityPub - Create Activity](https://www.w3.org/TR/activitypub/)
- [Decentralized Trust Graph - Membership Credential](https://github.com/trustoverip/dtgwg-cred-tf/blob/main/dtg.md)

Types of proofs:
- DataIntegrityProof with `ecdsa-rdfc-2019` cryptosuite
- DataIntegrityProof with `eddsa-rdfc-2022` cryptosuite
